# LognTikTok - Simulasi Login Edukatif

⚠️ Proyek ini hanya untuk edukasi. Tidak digunakan untuk mencuri data atau merugikan pihak manapun.

## Cara Menjalankan
- Jalankan `python contohphissing.py`
- Buka `http://127.0.0.1:5000` (lokal) atau Web Preview Replit

## Fitur
- Halaman login mirip TikTok
- Menyimpan login palsu ke `log.txt`
- Viewer log via `/log`
